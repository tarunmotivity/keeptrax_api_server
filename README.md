Keeptrax_api_server is mainly written to maintain the manager user relationship 


Node version is 16.15.0

**************************************
step 1: Install NVM (Node Version Manager) to maintain multiple node verions

step 2: Install node using bellow command

        nvm install 16.15.0

step 3: Use Installed node version using bellow command

        nvm use 16.15.0

step 4: Run the project using this command

        node server.js
